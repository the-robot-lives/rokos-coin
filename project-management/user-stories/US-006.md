# US-006: View my miner dashboard with earnings and uptime stats

**As a** independent miner and compute provider,
**I want to** view a real-time dashboard showing my earnings, uptime percentage, job completion rate, and current stake,
**So that** I can monitor my node's health and profitability and take action if performance drops.

## Acceptance Criteria
- [ ] Dashboard displays total ROKO earned (all time), pending claimable balance, and current staked amount
- [ ] Uptime percentage is calculated over rolling 24h, 7d, and 30d windows and displayed with trend indicator
- [ ] Jobs served count and success/failure ratio are shown with breakdown by model/tier served
- [ ] Earnings history is displayed as a time-series chart with at least 30-day granularity
- [ ] Dashboard data refreshes automatically (polling or WebSocket) without requiring a manual page reload

## Notes
Data sourced from on-chain events indexed by a subgraph or backend indexer. Miner-specific data should be scoped to the connected wallet. Consider surfacing rank on leaderboard (US-022) and any active slashing warnings.
