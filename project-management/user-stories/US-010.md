# US-010: View real-time token supply and emission rate

**As a** DeFi investor and token holder,
**I want to** view the current circulating supply, total supply, and live emission rate of ROKO,
**So that** I can assess inflationary pressure, model token value trajectories, and make informed investment decisions.

## Acceptance Criteria
- [ ] A public stats page or API endpoint exposes: total supply, circulating supply, locked/staked supply, and current epoch emission rate
- [ ] Emission rate is shown as ROKO per epoch and annualized percentage relative to circulating supply
- [ ] Historical emission data is available as a downloadable CSV or chart with at least 90-day history
- [ ] Circulating supply calculation excludes tokens locked in treasury, vesting contracts, and staking — these are itemized separately
- [ ] Data updates in near real-time (within one epoch or block confirmation) without requiring a page refresh

## Notes
Emission rate may be dynamic based on network utilization or governance votes. The stats endpoint should be designed for external consumption by CoinGecko/CoinMarketCap listing requirements. Token burn events from compute consumption should also be surfaced here.
