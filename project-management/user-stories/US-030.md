# US-030: View the vesting schedule for team/investor allocations

**As a** DeFi investor and token holder,
**I want to** view the exact vesting schedule for team, founder, and investor token allocations,
**So that** I can model future sell pressure from vesting unlocks and assess whether the team is genuinely committed for the long term.

## Acceptance Criteria
- [ ] A public vesting page shows each allocation bucket with: total allocated, cliff date, linear vesting end date, tokens vested to date, and tokens remaining locked
- [ ] Vesting contract addresses are published and linked, allowing independent verification of locked balances
- [ ] Upcoming unlock events are displayed as a calendar or timeline — at minimum the next 12 months of scheduled unlocks
- [ ] If any team or investor allocation has been modified from the originally published schedule, the change is disclosed with the governance proposal or board decision reference
- [ ] A "vesting unlock" alert or notification service allows investors to subscribe to upcoming unlock event reminders

## Notes
Vesting transparency is a trust foundation for any token project. Undisclosed or hidden allocations are a major red flag for the DeFi community. The vesting contracts should be immutable once deployed — no admin function to accelerate vesting — and this should be independently audited and disclosed.
