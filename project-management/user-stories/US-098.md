# US-098: Browse the Compute Marketplace with Filters and Sorting

**As an** enterprise architect,
**I want to** browse the ROKO compute marketplace with rich filtering and sorting capabilities,
**So that** I can identify miners that meet my organization's requirements for model type, performance tier, geographic region, and SLA guarantees before committing resources.

## Acceptance Criteria
- [ ] Marketplace listing page displays all active miner offers with: capability category, GPU model, VRAM, availability (live/queued), price per request, reliability score, response latency (p50/p99), and stake amount
- [ ] Filters available: capability category (multi-select), minimum reliability score (slider), maximum price per request, GPU type, geographic region (if disclosed), and minimum stake tier
- [ ] Sorting options: price ascending/descending, reliability score, response latency, stake amount, newest registered
- [ ] Each miner card links to a detail page showing: full capability list, uptime history chart, past verification results, stake history, and a "Book Now" / "Add to Comparison" action
- [ ] Comparison view allows side-by-side comparison of up to 4 miner profiles across all metrics

## Notes
Geographic region data is voluntary and unverified at protocol level; display with a disclaimer. Reliability scores should link to their calculation methodology to build trust with enterprise evaluators. Consider a "Verified Enterprise" tier badge for miners who have passed additional KYC or SLA certification processes. Pagination or infinite scroll is required for large miner sets.
