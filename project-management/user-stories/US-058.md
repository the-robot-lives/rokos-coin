# US-058: View Real-Time Marketplace Supply/Demand Metrics

**As a** Governance Council (DAO),
**I want to** monitor real-time supply and demand metrics across the ROKO marketplace,
**So that** the council can make informed decisions about protocol parameters such as floor prices, miner incentives, and capacity incentives.

## Acceptance Criteria
- [ ] A public governance analytics dashboard displays: active miners by hardware tier, total available capacity (tokens/sec), current request queue depth, current spot price per 1K tokens by model, and 24h request volume
- [ ] Historical charts show supply and demand trends over 7d/30d/90d windows with the ability to overlay governance parameter changes as annotations
- [ ] Dashboard includes a "supply gap" indicator showing models or regions where consumer demand consistently exceeds available miner supply, flagging them as incentive opportunities
- [ ] All metrics are derived from indexed on-chain events; the data pipeline lag is under 5 minutes and the dashboard displays a staleness warning if the indexer falls behind
- [ ] Governance council members can export raw metric data as CSV for off-chain analysis and proposal drafting

## Notes
This dashboard serves a dual audience: governance members making protocol decisions, and prospective miners evaluating whether to join the network. The supply gap indicator (criterion 3) directly informs incentive design — governance can increase mining rewards for underserved model categories.
