# US-046: Browse Available Compute Capabilities on the Marketplace

**As a** enterprise architect,
**I want to** browse the full catalog of compute capabilities available on the ROKO marketplace,
**So that** I can assess whether the network meets my organization's model, hardware, and throughput requirements before committing budget.

## Acceptance Criteria
- [ ] The marketplace UI displays a searchable, paginated catalog of available compute offerings, each showing: supported models, hardware tier (GPU type/VRAM), average latency (p50/p95), current price per token, uptime percentage, and miner reputation score
- [ ] Catalog entries are grouped by capability category: LLM inference, image generation, speech-to-text, embeddings, and custom/fine-tuned models
- [ ] Each listing links to a miner profile page with full performance history, stake amount, and active SLA commitments
- [ ] Catalog data refreshes at most every 5 minutes and displays a "last updated" timestamp so users know data freshness
- [ ] A "compare" feature allows selecting up to 5 listings side-by-side with a normalized capability matrix

## Notes
Enterprise evaluation often happens before any wallet connection. The catalog should be fully browsable without signing in. Aggregated statistics across multiple miners offering the same model should be surfaced as a "network average" row at the top of each model group.
