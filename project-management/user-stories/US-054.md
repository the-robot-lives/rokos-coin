# US-054: Access Academic Discount Tier for Research Workloads

**As a** AI researcher,
**I want to** apply for and use an academic discount tier on ROKO compute costs,
**So that** my institution can afford to run experiments on the network without the full commercial rate straining our research budget.

## Acceptance Criteria
- [ ] An `AcademicRegistry` contract stores verified academic credentials (institution domain, ORCID hash, or verifiable credential CID) linked to a wallet address
- [ ] Verified academic wallets receive a governance-defined discount (e.g., 30% off protocol fee) automatically applied at settlement for any request originating from the verified address
- [ ] Verification requires submitting a credential CID signed by a recognized academic verifier (e.g., on-chain oracle attesting .edu email ownership) — no centralized whitelist admin key
- [ ] Academic rate applies only to non-commercial use; on-chain usage data is sampled by governance to detect commercial API resale, which triggers discount revocation and potential stake penalty
- [ ] Academic discount usage and remaining period are visible in the researcher dashboard alongside standard usage metrics

## Notes
Academic discounts drive research adoption and generate papers/citations that improve protocol credibility. The non-commercial enforcement (criterion 4) is challenging — consider a self-attestation + stake-based accountability model rather than active monitoring. Credential verification should use a decentralized identity standard (e.g., W3C VC) where possible.
