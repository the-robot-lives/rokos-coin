# US-071: View Proposal History with Vote Tallies and Outcomes

**As a** Governance Council member,
**I want to** browse a complete history of all past governance proposals with their final vote tallies and execution status,
**So that** the DAO can maintain institutional memory, track precedent, and hold participants accountable.

## Acceptance Criteria
- [ ] Governance portal lists all proposals in reverse chronological order with ID, title, proposer address, final state, and vote counts
- [ ] Each proposal detail page shows: full description, calldata summary, voting timeline, per-address vote breakdown (address, vote, weight, reason), and execution transaction hash if applicable
- [ ] Proposals are filterable by state (Executed, Defeated, Expired, Pending, Active), category, and date range
- [ ] Historical data is indexed from on-chain events via a subgraph or equivalent indexer; page loads in under 3 seconds
- [ ] Defeated or expired proposals display the failure reason (quorum not met vs. majority Against)

## Notes
Consider linking each proposal to off-chain Discourse/forum discussion thread if applicable. Export to CSV for governance reporting. Proposal descriptions stored as IPFS CID should be resolved and displayed with a fallback message if IPFS is unavailable.
