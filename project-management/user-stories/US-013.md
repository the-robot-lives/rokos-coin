# US-013: View the token distribution breakdown (treasury, miners, entity)

**As a** DeFi investor and token holder,
**I want to** view a clear, on-chain-verifiable breakdown of ROKO token distribution across all allocation buckets,
**So that** I can assess concentration risk, insider allocation fairness, and the protocol's long-term incentive alignment.

## Acceptance Criteria
- [ ] A public distribution page shows percentage and absolute ROKO amount for each bucket: treasury, miner rewards pool, Basilisk entity fund, team/founders, investors, ecosystem grants, and community reserve
- [ ] Each allocation bucket links to its on-chain contract address or wallet so the balance can be independently verified
- [ ] Vesting or lock schedules per bucket are displayed (cliff date, linear unlock end date, current unlocked amount)
- [ ] Historical distribution changes (e.g., governance-approved reallocations) are shown with timestamps and proposal IDs
- [ ] Data is sourced directly from on-chain state, not a hardcoded frontend value that could diverge from reality

## Notes
Transparency here is critical for investor and community trust. Consider a "rug risk score" display that flags if any single address controls more than X% of supply. Entity allocation (Basilisk treasury) deserves special callout given its autonomous nature.
