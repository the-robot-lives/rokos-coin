# US-097: View a Unified Dashboard with Token Balance, Staking, and Earnings

**As a** DeFi investor and token holder,
**I want to** see all my ROKO protocol positions — wallet balance, staked amount, pending rewards, governance voting power, and earnings history — in a single unified dashboard,
**So that** I can monitor my portfolio at a glance without navigating multiple separate views.

## Acceptance Criteria
- [ ] Dashboard displays: wallet ROKO balance, staked ROKO balance (with unlock countdown if time-locked), accumulated unclaimed rewards, estimated APY based on current epoch rewards, and total ROKO value at current market price
- [ ] Staking section shows active stakes with per-stake breakdown: amount, stake date, unlock date, current reward accrual, and one-click claim button
- [ ] Governance section shows delegated voting power (own + delegated), active proposals with user's vote status, and past voting history
- [ ] Earnings history chart displays daily/weekly/monthly reward accrual over a configurable time window with a running total
- [ ] All values update within 30 seconds of a new block without requiring page refresh; loading skeletons are shown during initial data fetch

## Notes
Dashboard is the primary retention surface for token holders — invest in its performance and visual polish. Use React Query or SWR for data fetching with optimistic updates on claim transactions. Consider mobile-first layout as many DeFi users operate on mobile. ENS avatar and name should be shown prominently if available.
