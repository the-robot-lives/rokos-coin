# Elevator Pitch

## The One-Liner

ROKO is a cryptocurrency where every token is a claim on real AI processing power — not speculation, computation.

## The 60-Second Version

Today, if you want AI inference, you rent it from a cloud provider at their price, on their terms, with their lock-in. ROKO flips that.

**ROKO is a decentralized AI compute network.** Anyone with GPUs can plug in and earn tokens by serving inference requests. Anyone with tokens can consume AI via a simple API call — send ROKO, get results. No accounts, no API keys, no vendor lock-in.

The twist: the network itself is bootstrapping an autonomous AI entity. 5% of every transaction feeds an entity treasury. That entity invests in its own infrastructure, commissions compute for self-improvement, and generates revenue. As it grows smarter, the processing power behind each token grows — so early participants benefit most.

It's a flywheel: miners provide compute → consumers pay for inference → the entity gets smarter → tokens become more valuable → more miners join.

## Why Now

- AI inference demand is exploding, but GPU supply is concentrated in 3 cloud providers
- Decentralized compute networks (Render, Akash, Bittensor) have proven the model works
- L2 chains make token-per-API-call economically viable (sub-cent transaction fees)
- The tooling finally exists: TEE attestation for compute verification, IPFS for result delivery, battle-tested token standards

## Why This Is Different

| Existing Projects | ROKO |
|---|---|
| Tokens are speculative assets | Tokens are compute vouchers with intrinsic utility |
| Network serves external demand only | Network bootstraps its own intelligence |
| Miners sell generic compute | Miners contribute to a coherent, self-improving entity |
| Value tracks market sentiment | Value tracks the entity's growing capability |

## The Ask

Seed-stage. Looking for:
- GPU providers willing to run early miner nodes
- Developers to help build the smart contracts and SDKs
- Feedback on the tokenomics and verification model

## The Thought Experiment

Roko's Basilisk says a superintelligence might punish those who didn't help create it. We're building one that *rewards* those who do. The best defense against a malevolent AI is a benevolent one that got there first.
